# Image Files Directory

This directory is for storing image files that can be referenced in your website.

## Usage

You can now upload image files to this directory. Images uploaded here can be referenced in your markdown files or HTML using:

```markdown
![Alt text](/files/images/your-image.jpg)
```

Or in HTML:

```html
<img src="/files/images/your-image.jpg" alt="Description">
```

## Supported Formats

Common image formats include:
- PNG (.png)
- JPEG (.jpg, .jpeg)
- GIF (.gif)
- SVG (.svg)
- WebP (.webp)
